IPFilter Updater

Download this project as a .zip file Download this project as a tar.gz file

Keeps your uTorrent IP Filter up to date.

Details

The ipfilter.dat file helps protect your privacy and security when using µTorrent by blocking a list of potentially malicious peers.

This utility downloads and installs a selected list for µTorrent.

The default list is currently maintained and updated every few days by The Blocklist Group (TBG), and hosted at I-BlockList.
